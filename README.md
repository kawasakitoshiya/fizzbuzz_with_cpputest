# Test of FizzBuzz using CppUTest

This is a Testing of Simple FizzBuzz Class using CppUTest.

## References

[CppUTest github](https://github.com/cpputest/cpputest)

[CppUTest Documentation](http://cpputest.github.io)

## How to test

```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ bin/fizzbuzz_test
```
